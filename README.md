# ConvertHub

ConvertHub is a free online tool suite that allows users to seamlessly convert files directly in their browser. It features no watermarks, secure client-side only processing, and a beautiful UI.

## Features
- **Photo to PDF**
- **Word to PDF**
- **PDF to Word**
- **Image Compression**
- **Passport Photo Maker**
- **Image Cropping**
- ...and more!

## Running Locally

1. Install the dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. To build for production:
   ```bash
   npm run build
   ```

## Deployment
This project is configured and ready to be seamlessly deployed onto [Vercel](https://vercel.com). Because Vercel natively supports Vite, just connect this repository to your Vercel account, and it will deploy perfectly out of the box!
